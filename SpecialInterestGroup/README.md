# Special Interest Groups (SIGs)

This directory contains documentation and proposals for all **Special Interest Groups (SIGs)** under the Stony Brook Computing Society (SBCS).

Each SIG is a focused community within SBCS with its own leadership, meetings, and initiatives.

---

## Creating a new SIG

Per the SBCS Constitution, a new SIG must:

1. Be proposed by a student.  
2. Have:
   - A **name**
   - A **brief statement of purpose**
   - A **proposed SIG executive board**
3. Receive **Â¾ Executive Board approval**.  
4. Note: SIGs formed mid-semester may only request funding starting the **next semester**.

New SIG proposals should be added here as a markdown file.

### File layout

- `proposals/`  
  - New SIG proposals (one file per SIG)
- `active/`  
  - Documentation for approved, active SIGs
- `archive/`  
  - Retired or inactive SIGs

---

## Naming conventions

- Proposal files:  
  - `proposals/<sig-name>-proposal.md`  
  - Example: `proposals/builder-club-proposal.md`

- Active SIG files:  
  - `active/<sig-name>.md`  
  - Example: `active/builder-club.md`

---
